# Lab: SSH Brute Force Detection

## Objective
Simulate and detect unauthorized SSH login attempts.

---

## Tools Used
- Linux VM
- Nmap
- Auth logs
- Terminal

---

## Activity
Simulated repeated login attempts using incorrect credentials.

---

## Findings
Auth logs showed:
Failed password attempts from 192.168.x.x

Pattern indicates brute-force behavior.

---

## Response
Documented event and recommended IP blocking.

---

## Skills Practiced
- Log analysis
- Incident reporting
- Linux security review
